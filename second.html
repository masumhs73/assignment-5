Question-1: What are the different ways to select an element in the DOM?

There are several different ways to select an element in the Document Object Model (DOM), including

  • getElementById(id)
  • getElementsByClassName(className)
  • getElementsByTagName(tagName)
  • querySelector(selector)
  • querySelectorAll(selector)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property returns all text including html tags, contained by an element. On the other hand, the textContent property and the innerText property returns all the texts contained by an element and also it's child element.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique where a parent element handles events for its child elements, even if children are added dynamically after the page loads. It works because of JavaScript's events bubble up method from the target element to its ancestors.

Question-4: What is event bubbling in the DOM?

Event bubbling is the phase where the event bubbles up from the target element all the way to the global window object. Once reached the target element, the event enters the bubbling phase, propagating back up to the parent through the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

There are several different ways to select an element in the Document Object Model (DOM), including

  • Create an Element

    In JavaScript the document.createElement() method is used to create a new HTML element.

    Example:
    const newParagraph = document.createElement('p');
    p.innerText = "Hello, I am a full stack Web Developer.";

  • Add Element to the Page

    After creating an element, it can be added to the DOM using appendChild() method. It adds the new element as the last child of the specified parent element.

    Example:
    const container = document.getElementById('parent-container');
    container.appendChild(newParagraph);

  • Remove an Element

    We can use removeChild() method on the parent node to remove an element, passing the element to be removed as an argument. On another way, we can use remove() method directly on the element to be removed.

    Example:
    //Using removeChild()
    const parentElement = document.getElementById('parent');
    const childElement = document.getElementById('child');
    parentElement.removeChild(childElement);
    //Using remove()
    const removeElement = document.getElementById('removeElement');
    removeElement.remove();
